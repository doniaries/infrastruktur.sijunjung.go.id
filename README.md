cara cloning

1. git clone https://github.com/doniaries/laporan-gangguan-jaringan.git
2. buat database laporan-jaringan, ubah env
3. buka di vs code
4. composer install
5. php artisan key:generate
6. php artisan filament:user
7. php artisan filament:upgrade
8. npm install && npm run build
9. php artisan migrate
10. jika ada error permission, buka filament shield
11. php artisan shield:install
12. php artisan serve
13. http://127.0.0.1:8000/admin/
14. login dengan akun admin
15. untuk database tabel opds, kopikan isi tabel dari tabel opd ke opds dengan cara INSERT INTO opds SELECT * FROM opd di sql
